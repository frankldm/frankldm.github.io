Some advises for Dennis' travel to China

Travel Guide

There is english website for foreigner travellers.
http://www.travelchinaguide.com/

  • Apply for a QQ account and a Weixin account, it is the most popular application in China.
  • Emergency call: if you get robbed or your things is lost, call 110. for medical emergency, call 120.
  • There are several websites for tourists to search the real local things.

    www.couchsurfing.com

    www.douban.com // you need to register //

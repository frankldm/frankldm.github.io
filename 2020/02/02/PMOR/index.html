Parametrized Model Order Reduction using Moment-Matching toolbox

A parametrized state-space, model order reduction toolbox developed at Control Systems Group, Eindhoven University of Technology.

Copyright

  • This toolbox is developed by Daming Lou (https://research.tue.nl/en/persons/daming-lou), PhD candidate at (https://www.tue.nl/en/research/research-groups/control-systems/).
  • This project is sponsored by Advanced Thermal Control (ATC) Consortium(consortium).

Note: we offer limited access to our industrial partners and academic reseachers. For obating the access to this repository, please contact the toolbox developer via: pmorhelp@tue.nl

Compatibility

  • Developed in MATLAB 2017a.
  • Tested on: MATLAB 2013a, MATLAB 2013b, MATLAB 2018a (Windows 7, Windows 10 and MacOS 10), check the changelog

Requirements

  • For some specific functions, control system toolbox is needed. (https://www.mathworks.com/products/control.html)
  • The ongoing development integrated with optimization requires optimization toolbox. (https://nl.mathworks.com/products/optimization.html)

Benchmark

  • The present benchmark characterizes a thermal/heat conduction in solids and the thermo-induced deformation.
  • The thermal simuluation is governed by time-dependent heat equation using Fourier’s law; the deformation is given by a static mapping.
  • The benchmark file is provided with .mph COMSOL (v5.3 & 5.4).

Acknowledegement

The developing team (only one) is thankful to all the suggestions and feedback from industrial users, that they have contributed at providing industry-approved test cases and evaluations methods.

To be able to handle large-scale (n>>10^4) state-space computation, we use the function developed by sssMOR (http://www.rt.mw.tum.de/?sssMOR). We sincerely acknowledge and appreciate for their fantistic work.

Folder structure

The folder structure of the toolbox is as follows

  • pMOR(main folder)

    • doc
      -Manual and presentation slides

    • benchmark
      -A simple thermal model extracted from industrial partners

    • src(source code)
      -extras (extra functions used for large-scale transfer function evaluation)
      -MOR (reduction algorithm)
      -pssExtraction (parametric state-space decoupling/extraction algorithm)
